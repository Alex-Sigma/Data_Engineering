# Iris ML Pipeline (Python Only)

This project processes the classic Iris dataset with full machine learning logic in pure Python.

## Structure

- `prc/`: core logic (features, training)
- `data/`: input and output data
- `test_fun/`: test scripts
- `ml/variants_train_model/`: alternative versions
- `dags/`: Airflow DAG (later)

## Setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
